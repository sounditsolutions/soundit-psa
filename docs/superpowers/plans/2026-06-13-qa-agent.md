# QA Agent + Harness Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up an autonomous QA agent (`soundit-psa/gastown.qa`) that drives the dev server in a real browser through a versioned scenario library (tickets + wiki first), filing bug/UX/doc-gap findings as labeled beads plus a run report — and ship the companion mine-on-Resolved trigger change the same effort exposed.

**Architecture:** Phase 1 is a self-contained PSA code change (mine on Resolved/Closed). Phases 2–5 build the QA capability: a managed dev queue worker, a Playwright/Chromium harness with a hard dev-host guard, the `gastown.qa` town agent with a findings pipeline (beads + markdown report), and the first scenario library + supervised run. Unit-testable code (guard, scenario parser, finding→bead mapper, report generator, the trigger change) is TDD'd; browser/Chromium install and town-agent provisioning are ops-setup with explicit verification gates and a documented MCP-vs-script-runner decision spike.

**Tech Stack:** Laravel 12 / PHP 8.3 (PSA app + the harness code lives in `app/Services/Qa/` + an artisan command), Node + Playwright (headless Chromium), Gas Town agent templates, beads. PHPUnit on SQLite `:memory:` for the PHP units; Playwright runs only against the live dev server (not in the PHPUnit suite).

**Spec:** `docs/superpowers/specs/2026-06-13-qa-agent-design.md` (read it first — §3 architecture, §4 companion change, §5 scenarios, §6 dev env).

**Branch:** create `feat/qa-agent` off `main` in a worktree (superpowers:using-git-worktrees).

**Conventions:** logic in services (`app/Services/Qa/`), thin artisan commands, `\RuntimeException` for guard violations, Pint before each commit, TDD for the PHP units. The QA agent itself is a Gas Town artifact, not PSA app code — its template lives under the town's agent config, not the repo.

```markdown
**Decision:** The mining trigger fires when a ticket reaches Resolved OR Closed with a non-empty resolution, and also when the resolution is added/edited while already in a terminal status. namespace App\Services\Qa;
```

```php
class QaTargetGuard
{
    /** @param array<int,string> $allowedHosts */
    public function __construct(private readonly array $allowedHosts) {}

    /** Returns the URL if its host is an allowed dev host; throws otherwise. */
    public function assertAllowed(string $url): string
    {
        $host = parse_url($url, PHP_URL_HOST);
        if ($host === null || $host === false || $host === '') {
            throw new \RuntimeException("QA target rejected — unparseable URL: {$url}");
        }
        if (! in_array($host, $this->allowedHosts, true)) {
            throw new \RuntimeException("QA target rejected — host '{$host}' is not an allowed dev host. QA never runs against non-dev hosts.");
        }

        return $url;
    }
}
```

```php
<?php

namespace App\Services\Qa;

class QaScenarioLoader
{
    /** @return array<int,QaScenario> */
    public function loadDir(string $dir): array
    {
        $out = [];
        foreach (glob(rtrim($dir, '/').'/*.md') ?: [] as $file) {
            $out = array_merge($out, $this->parse((string) file_get_contents($file)));
        }

        return $out;
    }

    /** @return array<int,QaScenario> */
    public function parse(string $markdown): array
    {
        $blocks = preg_split('/^## /m', $markdown);
        $scenarios = [];
        foreach ($blocks as $block) {
            $block = trim($block);
            if ($block === '' || ! preg_match('/^([a-z0-9-]+):\s*(.+)$/m', $block, $head)) {
                continue;
            }
            $scenarios[] = new QaScenario(
                id: $head[1],
                title: trim($head[2]),
                goal: $this->field($block, 'goal'),
                setup: $this->field($block, 'setup'),
                steps: $this->list($block, 'steps'),
                expectations: $this->list($block, 'expect'),
                watchFors: $this->list($block, 'watch'),
            );
        }

        return $scenarios;
    }

    private function field(string $block, string $key): string
    {
        return preg_match('/^-\s*'.preg_quote($key, '/').':\s*(.+)$/m', $block, $m) ? trim($m[1]) : '';
    }

    /** @return array<int,string> */
    private function list(string $block, string $key): array
    {
        // Capture the lines indented under "- <key>:" until the next top-level "- " field.
        if (! preg_match('/^-\s*'.preg_quote($key, '/').':.*$((?:\n(?:\s{2,}|\s*\d+\.).*)*)/m', $block, $m)) {
            return [];
        }
        $items = [];
        foreach (explode("\n", $m[1]) as $line) {
            $line = trim($line);
            $line = preg_replace('/^(\d+\.|-)\s*/', '', $line);
            if ($line !== '') {
                $items[] = $line;
            }
        }

        return $items;
    }
}
```

```php
<?php

namespace App\Services\Qa;

class QaFinding
{
    public const KINDS = ['bug', 'ux', 'docs'];

    /** @param array<int,string> $steps */
    public function __construct(
        public readonly string $scenarioId,
        public readonly string $title,
        public readonly string $kind,
        public readonly string $severity,
        public readonly array $steps,
        public readonly string $expected,
        public readonly string $actual,
        public readonly ?string $screenshotPath = null,
    ) {}

    public function dedupKey(): string
    {
        return $this->scenarioId.'|'.$this->title;
    }

    public function body(): string
    {
        return "**Scenario:** {$this->scenarioId}\n**Severity:** {$this->severity}\n\n"
            ."**Steps to reproduce:**\n".collect($this->steps)->map(fn ($s, $i) => ($i + 1).". {$s}")->implode("\n")."\n\n"
            ."**Expected:** {$this->expected}\n**Actual:** {$this->actual}\n"
            .($this->screenshotPath ? "\n**Screenshot:** {$this->screenshotPath}\n" : '')
            ."\n_Filed by gastown.qa. dedup-key: `{$this->dedupKey()}`_";
    }
}
```

`app/Services/Qa/QaFindingFiler.php`:
```php
<?php

namespace App\Services\Qa;

class QaFindingFiler
{
    /**
     * @param callable(array<int,string>):string $runner   executes a gc command, returns stdout (the new bead id on create)
     * @param callable():array<int,array{id:string,dedup_key:string}> $openFindings  lists open qa beads with their dedup keys
     */
    public function __construct(
        private $runner,
        private $openFindings,
    ) {}

    public function file(QaFinding $finding): string
    {
        if (! in_array($finding->kind, QaFinding::KINDS, true)) {
            throw new \InvalidArgumentException("Invalid finding kind '{$finding->kind}'. One of: ".implode(', ', QaFinding::KINDS));
        }

        foreach (($this->openFindings)() as $open) {
            if (($open['dedup_key'] ?? null) === $finding->dedupKey()) {
                return $open['id']; // already filed — don't duplicate
            }
        }

        $cmd = [
            'gc', 'bd', 'create', '--rig', 'soundit-psa',
            $finding->title, '-t', 'task',
            '--label', 'qa', '--label', $finding->kind,
            '-d', $finding->body(),
            '--json',
        ];
        $out = trim(($this->runner)($cmd));

        // bd create --json returns the bead; accept either a raw id or a JSON object with .id
        $decoded = json_decode($out, true);

        return is_array($decoded) && isset($decoded['id']) ? $decoded['id'] : $out;
    }
}
```

```php
<?php

namespace App\Services\Qa;

class QaRunReport
{
    /**
     * @param array<int,array{scenario:string,status:string}> $results
     * @param array<int,array{id:string,kind:string,title:string}> $findings
     */
    public function __construct(
        private readonly string $runLabel,
        private readonly array $results,
        private readonly array $findings,
    ) {}

    public function toMarkdown(): string
    {
        $counts = ['pass' => 0, 'fail' => 0, 'error' => 0];
        foreach ($this->results as $r) {
            $counts[$r['status']] = ($counts[$r['status']] ?? 0) + 1;
        }
        $summary = "{$counts['pass']} pass · {$counts['fail']} fail · {$counts['error']} error";

        $lines = ["# QA run — {$this->runLabel}", '', "**Summary:** {$summary}", '', '## Scenarios'];
        foreach ($this->results as $r) {
            $icon = ['pass' => '✓', 'fail' => '✗', 'error' => '⚠'][$r['status']] ?? '?';
            $lines[] = "- {$icon} {$r['scenario']} ({$r['status']})";
        }
        $lines[] = '';
        $lines[] = '## Findings';
        if ($this->findings === []) {
            $lines[] = 'No findings.';
        } else {
            foreach ($this->findings as $f) {
                $lines[] = "- [{$f['kind']}] {$f['title']} ({$f['id']})";
            }
        }

        return implode("\n", $lines)."\n";
    }
}
```

Signature `qa:report {resultsJsonPath} {--mail=}`. (Mailing reuses the app's existing mail path; if none is trivially available, save-only is acceptable for v1 — note it.)
